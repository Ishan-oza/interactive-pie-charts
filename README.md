# ü•ß Python Interactive Pie Chart Generator

This is a simple Python script that generates **pie charts** using user input.  
Instead of writing new code every time you want to make a pie chart, this script lets you just **enter labels and values interactively**, and it will plot the chart for you.  

---

## ‚ú® Features
- Input custom labels and values directly in the terminal.
- Option to add **explode effect** (slice separation).
- Option to display **percentages** on the chart.
- Customizable chart title.
- Built with `matplotlib`.
